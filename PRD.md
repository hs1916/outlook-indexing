# PRD — Outlook PST 인덱서 & 메일 검색 시스템

**버전**: 4.0 (최종)
**작성일**: 2026-04-20
**작성자**: @planner

---

## 1. 제품 개요

### 배경
업무망 내에 보관된 대용량 Outlook PST 파일(5~10 GB)에서 특정 메일을 찾으려면 Outlook을 직접 열어 수동 탐색해야 하며, 파일 수가 많을수록 시간이 오래 걸린다.

### 해결 목표
JAR 파일 하나를 실행하면 로컬 웹 서버가 뜨고, 브라우저에서 PST 인덱싱과 메일 검색을 모두 사용할 수 있는 **오프라인 웹 서비스**를 제공한다.

### 실행 방식
```
java -jar pst-search.jar
→ 브라우저에서 http://localhost:8080 접속
→ 모든 기능 사용 가능
```

### 대상 사용자
- Windows PC 사용자 (개인)
- 업무망(인터넷 단절) 환경에서 사용
- JRE 17 이상이 설치된 PC

## 2. 핵심 기능 (화면 2개)

---

### 기능 1. PST 파일 & 인덱스 관리 화면

**사용자 스토리**
나는 업무망 사용자로서, 관리 화면에서 PST 파일을 등록하고 인덱싱을 실행하여, 어떤 파일이 인덱싱되어 있는지 한눈에 파악하고 관리할 수 있기를 원한다.

**수용 기준**
- [x] PST 파일 경로를 직접 입력하거나, **Windows 파일 탐색기 다이얼로그**를 통해 선택하여 등록할 수 있다
- [x] 등록된 PST 파일 목록을 테이블로 표시한다 (파일명, 경로, 크기, 인덱싱 상태, 마지막 인덱싱 일시)
- [x] 파일별로 "인덱싱 시작" 버튼을 제공하며, 실행 시 진행률(%)과 처리 중인 메일 수를 실시간으로 표시한다
- [x] 인덱싱 완료 후 해당 파일의 총 메일 수와 소요 시간을 표시한다
- [x] 인덱싱 상태는 미인덱싱 / 진행중 / 완료 / 오류 4가지로 구분하여 표시한다
- [x] 등록된 PST 파일을 목록에서 삭제(인덱스 데이터 포함 제거)할 수 있다
- [x] 이미 인덱싱된 파일을 재인덱싱하면 기존 데이터를 덮어쓴다
- [x] 파싱 중 오류가 발생한 메일은 스킵하고 계속 진행하며, 오류 건수를 완료 후 표시한다

### 기능 2. 메일 검색 화면

**사용자 스토리**
나는 업무망 사용자로서, 검색 화면에서 조건을 입력하고 검색하면 인덱싱된 메일 중 조건에 맞는 결과를 좌측 목록으로 확인하고, 선택한 메일의 전체 내용을 우측 패널에서 바로 볼 수 있기를 원한다.

#### 2-1. 검색 조건 (리본)

- [x] 검색 조건 영역은 화면 상단에 리본(Ribbon) 형태로 위치하며, 토글 버튼으로 펼침/접힘 전환이 가능하다
- [x] 다음 8가지 조건을 단독 또는 조합하여 검색할 수 있다:
  - 제목 키워드 (부분 일치)
  - 본문 키워드 (부분 일치)
  - 보낸사람 이름 (부분 일치)
  - 보낸사람 이메일 (부분 일치)
  - 받는사람 이름 (부분 일치)
  - 받는사람 이메일 (부분 일치)
  - 날짜 범위 (시작일 ~ 종료일)
  - 첨부파일명 키워드 (부분 일치)
- [x] 초기화 버튼으로 모든 조건을 한 번에 지울 수 있다

#### 2-2. 검색 결과 목록 (좌측 패널 ~1/4)

- [x] 검색 결과를 좌측 패널에 카드형 목록으로 표시한다 (제목 / 보낸사람 이름 / 날짜 / 첨부 유무)
- [x] 검색 결과가 없을 경우 안내 메시지를 표시한다
- [x] 검색 결과는 날짜 기준 내림차순(최신순)으로 정렬된다
- [x] 결과가 많을 경우 페이지네이션을 제공한다 (페이지당 50건)
- [x] 검색된 전체 메일 본문을 하나의 TXT 파일로 내보내기할 수 있다

#### 2-3. 메일 내용 보기 (우측 패널 ~3/4)

- [x] 목록에서 메일을 클릭하면 우측 패널에 메일 내용이 인라인으로 표시된다 (별도 팝업 없음)
- [x] 메일 상세에는 제목, 보낸사람(이름+이메일), 날짜, 받는사람(이름+이메일), 첨부파일 목록, 본문을 표시한다
- [x] 본문은 HTML 형식으로 렌더링한다 (표, 볼드, 색상 등 서식 보존)
- [x] 본문 내 인라인 이미지(CID 참조)가 우측 패널 안에서 바로 표시된다

#### 2-4. 첨부파일 다운로드

- [x] 우측 패널 헤더 영역에서 첨부파일을 개별 다운로드할 수 있다
- [x] 첨부파일이 2개 이상인 경우 "전체 ZIP" 버튼으로 일괄 다운로드할 수 있다

#### 2-5. 패널 크기 조절

- [x] 좌측 목록과 우측 내용 사이의 구분선을 드래그하여 각 패널의 너비를 자유롭게 조절할 수 있다 (15%~60% 범위)
- [x] 화면 전체가 브라우저 뷰포트 크기에 반응형으로 맞춰진다

## 3. 비기능 요구사항

| 항목 | 기준 |
|------|------|
| 실행 방식 | `java -jar pst-search.jar` 명령 하나로 서비스 구동 |
| 접속 방법 | 브라우저에서 `http://localhost:8080` 접속 |
| 인덱싱 속도 | 1GB PST 기준 10분 이내 처리 |
| 검색 응답 | 검색 실행 후 3초 이내 결과 표시 |
| 안정성 | 파싱 오류 발생 시 해당 메일 스킵 후 계속 진행 (전체 중단 없음) |
| 오프라인 동작 | 외부 인터넷 연결 없이 모든 기능 동작 |
| 호환성 | Windows 10/11, JRE 17 이상 |
| 저장 방식 | JAR 파일과 동일 경로에 SQLite DB 파일 자동 생성 |
| 저장 공간 | 인덱스 DB 크기는 원본 PST의 30% 이하 목표 |

## 5. 성공 기준

- `java -jar pst-search.jar` 실행 후 브라우저에서 즉시 사용 가능하다
- 10 GB PST 파일 인덱싱이 오류 없이 완료된다
- 8가지 조건 조합 검색 결과가 3초 이내 표시된다
- JRE가 설치된 Windows PC에서 JAR 파일 하나로 동작한다
- 메일 본문의 HTML 서식(표/이미지/볼드/색상)이 우측 패널에서 올바르게 표시된다
- 첨부파일 개별/ZIP 다운로드 및 검색결과 TXT 내보내기가 동작한다

## 6. Windows 시작 프로그램 등록

PC 부팅 시 자동으로 서버가 실행되도록 Windows 시작 프로그램에 등록한다.

### 방법 A — 시작 프로그램 폴더 (권장, 설치 불필요)

1. `Win + R` → `shell:startup` 입력 → 시작 프로그램 폴더 열기
2. 아래 내용으로 `.bat` 파일을 작성하여 해당 폴더에 저장

```bat
@echo off
start "" "C:\Program Files\Java\jdk-17\bin\javaw.exe" -jar "C:\pst-search\pst-search.jar"
```

> `javaw.exe` 를 사용하면 콘솔 창 없이 백그라운드로 실행된다.  
> JRE 경로와 JAR 경로는 실제 설치 위치에 맞게 수정한다.

3. 로그아웃/재부팅 후 자동 실행 확인
4. 브라우저에서 `http://localhost:8080` 접속

### 방법 B — 작업 스케줄러 (부팅 즉시 실행, 로그인 전에도 가능)

1. `Win + S` → "작업 스케줄러" 검색 → 실행
2. **작업 만들기** 클릭
3. **일반** 탭: 이름 `PST 메일 검색`, "사용자 로그온 여부에 관계없이 실행" 선택
4. **트리거** 탭: 새로 만들기 → "시작할 때" 선택
5. **동작** 탭: 새로 만들기
   - 프로그램/스크립트: `C:\Program Files\Java\jdk-17\bin\javaw.exe`
   - 인수 추가: `-jar "C:\pst-search\pst-search.jar"`
   - 시작 위치: `C:\pst-search\`
6. **조건** 탭: "AC 전원에 연결된 경우에만 실행" 체크 해제
7. 확인 후 관리자 암호 입력

### 방법 C — Windows 서비스 등록 (WinSW 사용)

[WinSW](https://github.com/winsw/winsw) 를 사용하면 JAR을 Windows 서비스로 등록할 수 있다.

1. `winsw.exe` 를 JAR과 같은 폴더에 복사
2. 같은 폴더에 `pst-search.xml` 작성:

```xml
<service>
  <id>pst-search</id>
  <name>PST 메일 검색 서버</name>
  <description>Outlook PST 인덱서 및 메일 검색 서비스</description>
  <executable>javaw</executable>
  <arguments>-jar "%BASE%\pst-search.jar"</arguments>
  <logmode>rotate</logmode>
</service>
```

3. 관리자 권한 명령 프롬프트에서:

```cmd
winsw.exe install   # 서비스 등록
winsw.exe start     # 서비스 시작
```

4. 이후 PC 재부팅 시 자동으로 서비스가 시작된다
5. 서비스 제거: `winsw.exe uninstall`

### 등록 방법 비교

| 방법 | 설치 난이도 | 로그인 전 실행 | 추천 대상 |
|------|-----------|-------------|---------|
| A. 시작 프로그램 폴더 | ★☆☆ 쉬움 | 불가 (로그인 후 실행) | 개인 PC |
| B. 작업 스케줄러 | ★★☆ 보통 | 가능 | 개인 PC (자동 로그인 환경) |
| C. Windows 서비스 | ★★★ 어려움 | 가능 | 항상 켜놓는 업무 PC |
